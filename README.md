# Sunday School Management System

A web-based system for managing Sunday School member information, registration, and administrative records.

## 🚀 Project Goal
The goal of this project is to build a simple system that helps Sunday School administrators store and access member information easily.

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB

## 📌 Features
- Member registration
- Store member personal information
- Unique member ID system
- Admin access to view member records
- Member status management (active/inactive)

## 📂 Project Structure
```
project-folder/
│
├── server.js
├── package.json
├── routes/
├── models/
├── controllers/
└── config/
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/sunday-school-system.git
```

2. Install dependencies

```
npm install
```

3. Run the server

```
node server.js
```
